## 帮助提示

```bash
用法: ux <command> [options]

命令：
  ux build    构建
  ux clean    删除构建、缓存目录
  ux init     新建脚手架
  ux lint     检查代码规范
  ux publish  发布
  ux start    打开本地开发服务器
  ux test     单元测试

通用选项:
  -h, --help     显示帮助信息                                             [布尔]
  -v, --version  显示版本号                                               [布尔]

```
